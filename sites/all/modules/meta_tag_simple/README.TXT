README META TAG SIMPLE

LOAD:
Load this module as you would any module. You can copy the files to 
example.com/sites/all/modules, then go to Modules in your site and enable it. 
Or you can click the '+Install new module link' on the Modules page and install 
it from within the site.


ENABLE:
To use Meta Tag Simple, you must enable it on the content type page. Go to 
'Structure > Content types', choose a content type, find the meta tags settings 
tab, click the enable radio, and save.

USE:
Once meta tags are enabled in a content type, you will find that option on the 
bottom left of the screen with the other options on the node edit form. 
Simply enter your tags and save.
